  • Maquetado web

    Uso HTML, CSS, XHTML, FXML y herramientas como Bootstrap, Bootswatch, PrimeFaces y Materialize.

    He desarrollado aplicaciones con frameworks como Spring Boot, Spring Security, JPA, Hibernate, JDBC, JSF, JavaFX, PrimeFaces, Laravel, JQuery, React, NodeJS, Ionic.

    Sé diseñar e implementar bases de datos relacionales y NoSQL, he utilizado: MongoDB, MySQL, PostgreSQL y Firebase.

    También he utilizado herramientas como Git y GitHub, Maven, NPM, Yarn, Composer, MySQL Workbench, Xampp, Payara, Docker, Postman.

    Además eso, sé implementar contenedores de Docker en una máquina virtual en forma de microservicios.

    Conozco y manejo la terminal de Linux, he usado sistemas operativos como Debian, Ubuntu, ArchLinux y Manjaro.
